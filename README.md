# GameStore

A modern game distribution platform with a clean, minimalist interface.

## Features

- Game Store
- Game Library
- User Profile
- News Section
- Friends & Messages
- Wishlist
- Settings

## Running the Project

1. Install dependencies:
```
npm install
```

2. Start development server:
```
npm run dev
```

3. Build for production:
```
npm run build:prod
```

## Tech Stack

- React
- Vite
- React Router
- Axios
- React Icons
